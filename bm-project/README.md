# Business Management System (BMS)

## Project Overview
The **Business Management System (BMS)** is a web-based platform connecting job seekers with companies, providing training programs and job opportunities, and simplifying applications digitally.

## Features
- Job posting and candidate application  
- Access to training programs  
- Candidate search and digital workflow  

## Technologies
- Frontend: Next.js, Tailwind CSS  
- Backend: Firebase Functions, Firestore  
- Deployment: Vercel  
- Version Control: GitHub
